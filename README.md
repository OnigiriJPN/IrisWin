# IrisWin
IrisWin

**「ウィンドウに、虹の彩りと透明感を。」
— Windows 11 のネイティブ API を極限までシンプルに。**

***IrisWin（アイリス・ウィン）は、Windows アプリケーションの「外観」をモダンに進化させるための .NET 向け軽量ライブラリです。
ギリシャ神話の虹の女神「イリス」の名を冠し、標準では扱いにくい タイトルバーの彩色、アクリル・マイカ効果による透過、そして ピクセル単位の正確なサイズ補正 を、わずか数行のコードで実現します。***
